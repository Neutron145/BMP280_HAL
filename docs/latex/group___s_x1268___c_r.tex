\doxysubsubsection{Подробное описание}
Coding Rate определяет скорость кодирования при методе Forward Error Correction. Чем выше скорость, тем сильнее помехоустойчивость, но ниже скорость передачи данных. В типичных условиях достаточно скорость 4\+\_\+5.
