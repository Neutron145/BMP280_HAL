За макросами объявляются функции, перед которыми пишутся блоки документации этих функций. ~\newline
 Ключевые слова Javadoc, используемые в документации\+: ~\newline
 brief \{описание\} -\/ краткое описание элемента ~\newline
 details \{описание\} -\/ детальное описание элемента ~\newline
 defgroup $<$\+ID$>$ \{Заголовок\} -\/ объявление группы ~\newline
 param $<$имя параметра$>$ \{описание параметра\} -\/ описание параметра функции ~\newline
 retval $<$имя$>$ \{описание\} -\/ описание возвращаемого значения ~\newline
\hypertarget{group__br_autotoc_md1}{}\doxysubsubsubsection{\texorpdfstring{Как пользоваться документацией к библиотеке?}{Как пользоваться документацией к библиотеке?}}\label{group__br_autotoc_md1}
Документация локальная, хранится в виде HTML страниц в папке docs. Открывается через файл index.\+html. ~\newline
 Разбита по модулям библиотеки, находящиеся в заголовке \textquotesingle{}Topics\textquotesingle{} и по файлам, находящимся в заголовке \textquotesingle{}Файлы\textquotesingle{}. ~\newline
