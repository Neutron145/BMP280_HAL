Подробное описание

Модуль для акселерометра и гироскопа LSM6DS33.

Модуль позволяет конфигурировать датчик, снимать измерения гироскопа и акселерометра, программно сбрасывать датчик.
Модуль работает с библиотеками LL и HAL. Для выбора библиотеки необходимо использовать макрос и при использовании LL добавить в проект модуль I2C LL.
Перед измерением данных необходимо инициализировать датчик, сконфигурировать его, после чего можно снимать измерения. Пример работы с датчиком:

float a[3];
float g[3];
float t;
LSM6DS33_init(hi2c1);
LSM6DS33_get_all_measure(a, g, &t);
}

Конфигурация режима работы датчика

Конфигурируется через бит ODR_XL регистра CTRL1_XL и бит ODR_G регистра CTRL2_G. Позволяет включать или выключать гироскопа и акселерометра и задавать частоту их работы. Акселерометр и гироскоп работают независимо друг от друга, каждый на своих частотах.

Аргументы
a_ODRЧастота работы акселерометра. Принимает значения макросов частоты и питания датчика.
g_ODRЧастота работы гироскопа. Принимает значения макросов частоты и питания датчика.
Возвращаемые значения
statusСтатус I2C после конфигурации режима работы датчика. Может быть HAL_OK, HAL_ERROR, HAL_BUSY
